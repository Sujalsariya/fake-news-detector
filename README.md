# Fake News Detection 📰🧠

This is a natural language processing (NLP) project developed by **Sujal Sariya** during the **RISE Internship Program by Tamizhan Skills**.

The project uses machine learning to classify news as either **FAKE** or **REAL** based on the article text.

---

## 📦 Technologies Used

- Python 3.12
- pandas
- scikit-learn
- TF-IDF vectorization
- Passive Aggressive Classifier
